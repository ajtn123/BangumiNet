# 贡献指南

如果你希望为 BangumiNet 项目作出贡献，你可以...

- [打开 Issue](https://github.com/ajtn123/BangumiNet/issues)
  - 报告 BUG
  - 提出建议
- [打开 Pull Request](https://github.com/ajtn123/BangumiNet/pulls)
  - 帮助修复 BUG
  - 实现新功能
- [参与讨论](https://github.com/ajtn123/BangumiNet/discussions)
- 或者以其他任何形式
